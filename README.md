# pgh-events

Gatsby front end for https://pgh.events

## Getting Started

1.  **Install Gatsby CLI.**

    ```shell
    npm install -g gatsby-cli
    ```

1.  **Clone this repo.**

    ```shell
    git clone git@github.com:ericsakmar/pgh-events.git
    ```

1.  **Install dependencies.**

    ```shell
    cd pgh-events
    npm install
    ```

1.  **Start it up!**

    ```shell
    gatsby develop
    ```
